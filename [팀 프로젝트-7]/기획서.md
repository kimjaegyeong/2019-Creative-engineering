# 2. 과제 요약
```
그냥 날씨만 알려주기에는 디자인과 편리성이 없어, RGB LED 리모컨과 날씨 나타내기의 아이디어를 융합했습니다.
현재 온도에 따라서 RGB LED의 색깔이 변해 현재 온도를 알려주는 아두이노를 만들겠습니다.
```
